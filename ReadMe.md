# Simple Python Calculator

## Basics:
- Select the appropriate math operator.
- Enter the numbers.
- The result will be displayed.

## Supported Operators:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Example:
- Select the operator: +
- Enter the first number: 5
- Enter the second number: 10
- The result is: 15
